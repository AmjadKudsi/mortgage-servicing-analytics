About this dashboard

Built on Freddie Mac's Loan-Level Disclosure data (February 2026 release). The data pipeline processes 11 raw files spanning origination years 2014–2024, loads them into a DuckDB analytical database, and powers every visualization through SQL queries. The dashboard pages use Tableau Public's in-canvas navigation buttons — click "Go to Risk Migration Dashboard" or "Go to Vintage Analysis Dashboard" within the visualization above.
